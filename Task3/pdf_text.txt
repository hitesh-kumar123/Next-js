Advanced Next.js + GraphQL Full-Stack Platform
— Task Description
Project Overview
Build a production-ready full-stack platform using Next.js and GraphQL with a strong focus on
scalability, performance, API architecture, authentication, real-time features, and modern frontend
development practices.
The platform should demonstrate:
• Advanced Next.js architecture
• GraphQL API integration
• Real-time functionality
• Authentication and authorization
• Complex frontend state management
• Performance optimization
• Modern UI/UX implementation
The goal of this task is to evaluate advanced frontend and full-stack engineering skills, API design
understanding, scalability thinking, and production-level architecture.
